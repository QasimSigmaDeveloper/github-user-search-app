# 🤖 GitHub User Search App

## 📌 Project Description
This is a web application built using TypeScript, HTML, and CSS that allows users to search GitHub profiles using the GitHub REST API and display user information dynamically in a clean UI.

---

## ✨ Features
- Search GitHub users by username
- Display user avatar and profile
- Real-time API data fetching
- Responsive UI design
- Error handling for no results

---

## 🛠️ Technologies Used
- HTML
- CSS
- TypeScript
- JavaScript Fetch API
- GitHub REST API

---

## 🚀 How to Run Project
1. Clone repository
2. Open project folder
3. Open `index.html` in browser

---

## 📡 API Used
https://api.github.com/users
